# Запуск API-сервера демографической аналитики

## Требования
- Python 3.9+
- Установить зависимости из `requirements.txt`

## Установка
1. Клонируйте репозиторий.
2. Поместите файлы данных в папку `data/`:
   - `cities_historical_2015_2024.csv`
   - `cities_forecast_2024_2039.csv` (опционально)
3. Установите зависимости:
   ```bash
   pip install -r requirements.txt
4. Запустите сервер: 
* в терминале - `uvicorn api:app --reload`, 
* в командной строке Windows - `python -m uvicorn api:app --reload`
5. Сервер будет доступен по адресу: http://127.0.0.1:8000
6. Интерактивная документация API: http://127.0.0.1:8000/docs
